== Description ==

This a cool Wordpress 3D Image Rotator. Simple to use for presenting your produtcts or just a very nice image rotator.
What you can do:
 - present a list of products
 - 3d image rotator
 - image gallery
 - content slider
 - and you'll find more . . .

Key Features:

 - Ajax image uploader
 - Content color picker
 - Full control of content size
 - auto-generated reflection
 - auto initialization
 - auto resize
 - circular flow (wrap around)
 - vertical flow
 - loading indicator
 - dynamic opacity
 - click to select control
 - mouse wheel control
 - keyboard control
 - drag 'n throw control
 - item as link
 - caption
 - 18 scrollbar colors

Notice:You need to have active fileinfo extension.(In php.ini replace ";extension=php_fileinfo.dll" with "extension=php_fileinfo.dll")

== Installation ==

1. Unpack the download-package
2. Upload all files to the /wp-content/plugins/ directory
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Go to page or post edit
5. Success! You're ready to start adding shortcode to your blog posts and pages.
